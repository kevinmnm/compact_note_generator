INSTRUCTION

This compact note generator was created to reduce redundant process for BV agents when creating HBS notes. You no longer have to input repeating information nor information that are irrelevant to BV department. It will also automatically generate a list of templates that's required for your specific case outcome. The final note format will be exactly same with the original template generator.

STEPS:

  1. Input your name in "Note Entered By" field. Your name will be saved. No need to re-write upon every refresh!
  2. Choose plan type from the dropdown. This will lock your choice to provide accurate template list.
  3. Input drug information.
  4. Choose your case outcome from dropdown menu. Cases that only require 'DETAILED FREE-HAND NOTE' will not be on the list.
  5. Provide all the information listed on the templates, as leaving any of them can affect your audit score.

IMPORTANT!

You MUST refresh the page after finishing each case by using blue "REFRESH ALL" button at the bottom of the page or "BV Compact Note Generator" title at the top of the page.

The Internet Explorer web browser will cache all of your input (this means browser will save all of your info, causing error when starting new case). As our IT department blocked access to adjust brower option, I've created those buttons to circumvent cache problem.
*If you are using Chrome browser, you can disregard this message*
